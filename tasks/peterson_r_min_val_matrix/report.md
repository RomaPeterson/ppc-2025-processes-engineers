# Отчёт по задаче: Поиск минимальных значений по столбцам матрицы

## Описание задачи
Разработать программу для поиска минимальных значений в каждом столбце матрицы размером N×N, 
где элементы матрицы задаются формулой: `a[i][j] = i * N + j + 1`.

## Реализация

### Алгоритм
- Для каждого столбца j находим минимальное значение среди всех элементов этого столбца
- Матрица заполняется по формуле: элемент в строке i и столбце j равен `i * N + j + 1`
- Минимальное значение в столбце j всегда равно `j + 1` (элемент первой строки)

### Технологии
- **MPI**: Параллельное распределение столбцов между процессами
- **SEQ**: Последовательная обработка всех столбцов

### Особенности реализации
- Распределение работы в MPI версии с использованием `MPI_Gatherv`
- Корректная обработка граничных случаев
- Проверка валидности входных данных

## Результаты
Программа успешно находит минимальные значения по столбцам для матриц различных размеров.
